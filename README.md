# DTF (Digital Transformation Flow) 시각화 도구 사용 가이드

비즈니스 프로세스와 액티비티를 시각적으로 분석하는 도구입니다.

---

## 📁 1. 시작하기: 파일 업로드

### 파일 선택하기
1. 화면 좌측 상단의 **"Excel 파일 선택"** 버튼을 클릭합니다
2. 준비된 DTF 엑셀 파일을 선택합니다
3. 파일이 자동으로 분석되어 그래프가 화면에 나타납니다

> 💡 **파일 형식**: 엑셀 파일의 1행은 무시되고, 2행이 컬럼명, 3행부터 데이터로 인식됩니다

---

## 🎯 2. 그래프 화면 이해하기

### 화면 구성
- **왼쪽**: 프로세스 그래프 (메인 화면)
- **오른쪽**: 탭 메뉴 (에러 리포트, MM 테이블, L4 카테고리)
- **위쪽**: 팀 필터, 검색창

### 노드(박스)의 의미
- **색상**: L4 카테고리별로 다른 색상 (범례는 우측 L4 탭에서 확인)
- **테두리 두께**: 일반 프로세스는 얇게, 시작 노드(선행이 없는 노드)는 두껍게 표시
- **크기**: 모든 노드는 동일한 크기

### 화살표(엣지)의 의미
- **파란색 화살표**: 정상적인 선행/후행 관계
- **빨간색 화살표 + "⚠ 양방향" 라벨**: 양방향 연결 오류 (순환 관계)

---

## 🖱️ 3. 그래프 조작하기

### 기본 조작법

#### 줌 인/아웃
- **마우스 휠**: 위로 스크롤(줌 인), 아래로 스크롤(줌 아웃)
- 원하는 위치에서 자세히 보거나 전체를 볼 수 있습니다

#### 화면 이동 (패닝)
- **빈 영역을 드래그**: 그래프 전체를 이동할 수 있습니다
- 노드가 없는 배경을 클릭한 채로 끌어서 원하는 위치로 이동하세요

#### 노드 이동
- **노드를 드래그**: 개별 노드를 원하는 위치로 이동할 수 있습니다
- 겹쳐 보이는 노드들을 정리할 때 유용합니다

#### 노드 상세 정보 보기
- **마우스를 노드 위에 올리기**: 상세 정보 툴팁이 자동으로 나타납니다
- 툴팁에는 다음 정보가 표시됩니다:
  - 리소스 정보 (필요인력, 필요기간, MM)
  - 정의 및 설명
  - 프로세스 오너부서, 산출물 등
  - 선행/후행 프로세스 목록
  - 파이프(|)로 구분된 값은 태그 형태로 표시

#### L6 액티비티 보기
- **L5 노드를 한 번 클릭**: 해당 프로세스의 세부 액티비티(L6)로 이동합니다
- **뒤로가기**: 좌측 상단의 화살표 버튼을 클릭하면 L5 그래프로 돌아갑니다

### 고급 조작법

#### L5 필터링 모드
특정 프로세스와 연결된 노드만 집중해서 보는 기능입니다.

1. **L5 노드를 더블클릭**합니다
2. 선택한 노드와 직접 연결된 선행/후행 프로세스만 표시됩니다
3. 화살표에 흐르는 애니메이션이 나타나 관계를 명확히 보여줍니다
4. **"전체 보기" 버튼**을 클릭하면 전체 그래프로 돌아갑니다

> 💡 **활용 팁**: 복잡한 그래프에서 특정 프로세스의 흐름을 파악할 때 유용합니다

---

## 🔍 4. 필터 및 검색 사용하기

### 팀 필터
특정 팀의 프로세스만 선택해서 볼 수 있습니다.

1. 우측 상단의 **"팀 선택"** 버튼을 클릭합니다
2. 드롭다운 메뉴에서 원하는 팀을 선택합니다 (여러 개 선택 가능)
3. **"적용"** 버튼을 클릭합니다
4. 선택한 팀의 프로세스만 그래프에 표시됩니다
5. **"초기화"** 버튼으로 전체 프로세스를 다시 볼 수 있습니다

### 텍스트 검색
프로세스 이름으로 검색할 수 있습니다.

1. 우측 상단의 **검색창**에 찾고 싶은 단어를 입력합니다
2. 입력한 단어가 포함된 노드가 **노란색으로 하이라이트** 됩니다
3. 대소문자 구분 없이 검색됩니다
4. 검색어를 지우면 하이라이트가 사라집니다

---

## ⚠️ 5. 에러 리포트 확인하기

데이터의 문제점을 자동으로 탐지하여 알려줍니다.

### 에러 리포트 열기
1. 우측 상단의 **"에러 리포트"** 탭을 클릭합니다
2. 검증 오류 목록이 테이블 형태로 표시됩니다
3. 각 오류는 색상으로 구분됩니다

### 오류 유형

| 오류 유형 | 색상 | 의미 |
|----------|------|------|
| **대소문자 오류** | 파란색 | ID를 입력할 때 대소문자가 실제 ID와 다른 경우 |
| **누락된 선행** | 주황색 | 선행 프로세스로 지정했지만 실제로 존재하지 않는 경우 |
| **누락된 후행** | 보라색 | 후행 프로세스로 지정했지만 실제로 존재하지 않는 경우 |
| **양방향 오류** | 빨간색 | A와 B가 서로를 선행/후행으로 참조하는 순환 관계 |
| **선행/후행 불일치** | 분홍색 | A의 후행이 B인데 B의 선행에 A가 없는 경우 |
| **입력 누락** | 노란색 | 선행/후행이 비어있지만 다른 프로세스에서 참조하는 경우 |
| **Self-Loop 오류** | 노란색 | 자기 자신을 선행 또는 후행으로 참조하는 경우 |

### 오류 위치로 이동하기
1. 에러 리포트 테이블에서 **오류 행을 클릭**합니다
2. 자동으로 그래프 탭으로 전환되고 해당 프로세스가 표시됩니다

---

## 📊 6. MM 테이블 사용하기

시작 노드(선행이 없는 프로세스)의 리소스 정보를 확인할 수 있습니다.

### MM 테이블 열기
1. 우측 상단의 **"MM 테이블"** 탭을 클릭합니다
2. 시작 노드들의 정보가 테이블로 표시됩니다

### 테이블 정렬하기
- **컬럼 헤더를 클릭**: 해당 컬럼 기준으로 정렬됩니다
- **한 번 클릭**: 오름차순 정렬
- **다시 클릭**: 내림차순 정렬
- **다시 클릭**: 정렬 해제

### 표시 정보
- **L5 프로세스명**: 프로세스 이름
- **L5 ID**: 프로세스 고유 ID
- **필요인력**: 필요한 인원 수
- **필요기간**: 소요 기간
- **MM**: Man-Month (인력 × 기간)
- **누적 MM**: 해당 프로세스부터 시작하여 최종 노드까지의 총 MM

## 💡 7. 유용한 팁

### 복잡한 그래프 다루기
1. **줌 아웃**으로 전체 구조를 먼저 파악하세요
2. **팀 필터**로 관심 있는 팀만 표시하세요
3. **L5 필터링 모드**(더블클릭)로 특정 프로세스 주변만 보세요
4. **검색 기능**으로 원하는 프로세스를 빠르게 찾으세요

### 데이터 수정이 필요한 경우
1. **에러 리포트**를 먼저 확인하세요
2. 오류를 엑셀 파일에서 수정하세요
3. 수정된 파일을 다시 업로드하세요

### 툴팁이 화면 밖으로 나가는 경우
- 걱정하지 마세요! 툴팁은 자동으로 위치를 조정하여 항상 화면 안에 표시됩니다
- 노드가 화면 가장자리에 있어도 툴팁을 볼 수 있습니다

### 엑셀 파일 작성 시 주의사항
- **여러 값 입력**: 파이프(|)로 구분하세요 (예: `회의|email|시스템`)
- **관계 입력**: 선행과 후행 중 한쪽만 입력해도 자동으로 양쪽에 반영됩니다
- **대소문자**: ID는 대소문자를 정확히 일치시켜 입력하세요

## ❓ 문제 해결

### 그래프가 표시되지 않아요
- 엑셀 파일의 2행이 컬럼명인지 확인하세요
- Level 컬럼에 L5 또는 L6 값이 있는지 확인하세요

### 노드들이 겹쳐서 보여요
- 노드를 **드래그**하여 원하는 위치로 이동하세요
- **줌 아웃**하여 전체를 보거나 **줌 인**하여 겹친 부분을 자세히 보세요

### 오류가 너무 많이 나와요
- **대소문자 오류**: ID를 정확히 입력했는지 확인하세요
- **양방향 관계**: 선행과 후행 중 한쪽만 입력하세요 (자동 동기화됨)
- **누락 오류**: 참조한 ID가 실제로 존재하는지 확인하세요
